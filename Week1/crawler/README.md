# Crawler 

* chinatimes : 為爬取中國時報的爬蟲程式
* newhead : 為新頭客新聞的爬蟲程式
* free_news : 為自由時報的爬蟲程式

# design

為了統一爬蟲格式，所有爬蟲程式爬取的定義都統一化了，方便大家讀取以及使用
